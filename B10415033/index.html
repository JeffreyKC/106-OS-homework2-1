🤔 功能是什麼？

We are thinking of creating an alarm prior an event occuring. It will ring the user to notify that such an event will occur. This idea came from using Google Calender and apps to manage time schedule.

😇 如何實作？

Create new system call for alarm and date. In the proc.c, find allocproc function and add alarm and date. In the sysproc.c, create sys_alarm function and sys_date function. In the trap.c, find trap function and add condition if the date is reached, the alarm will ring.

🐻 預期成效（怎麼用、誰用）

User will input the date of an event. The system will then process the date of event with the current time. It will notify user on the day before the event and notify user again on that day before the event. The expected users for this are people who have time management issues and people who have time constraint.

陳俊翰: will create the alarm function in xv6. Test and Debug will also be handled
張紹山: will create the date function in xv6. Test and Debug will also be handled.
